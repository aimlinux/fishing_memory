## 📖 Overview | 概要

> **"ただ魚を釣るんじゃない。記憶を刻むんだ。"**

『ANGLER'S LOGBOOK』は、釣りを愛し、釣りに人生を捧げるすべての人のためのアプリケーションです。
一瞬のドラマ、感動の記録、そして次なる大物への戦略。
あなたのフィッシングライフを、より深く、よりスマートにサポートします。

**2026年 春休み、熱き魂を持った開発者たちによって、わずか「一週間」で生み出されました。**

---

## ✨ Features | 主な機能

<div align="center">

| 機能 | アイコン | 説明 |
| :--- | :---: | :--- |
| **爆釣記録 (Catch Log)** | 📸 | 釣った魚の写真、サイズ、ルアー、天候を一瞬で記録。 |
| **ポイントマップ (Point Map)** | 🗺️ | 自分だけのシークレットポイントをGPSで管理。 |
| **潮汐・気象連動 (Tide & Weather)** | 🌊 | 現在地のタイドグラフと気象情報をリアルタイム表示。 |
| **タックル管理 (Tackle Box)** | 🎣 | 愛用のロッドやリール、ルアーのコレクションをデジタル管理。 |

</div>
